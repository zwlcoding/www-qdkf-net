# 渊渟

> 执经衡门，渊渟岳立

## 项目介绍

渊渟是 waylon 的独立创作者主页，用于记录个人思考与项目实践。网站采用极简设计风格，以深色主题为主，突出沉静、克制、长期主义的个人表达。

## 技术栈

- **HTML5**: 语义化标签构建页面结构
- **CSS3**: 内联样式，无需额外依赖
- **纯静态网站**: 无需后端服务，部署简单

## 项目结构

```
.
├── index.html          # 首页（唯一页面）
├── ads.txt             # 广告验证文件
└── README.md           # 项目说明文档
```

## 部署方式

由于本项目是纯静态网站，可通过以下方式部署：

### 方式一：直接上传
将 `index.html` 文件上传至任意 Web 服务器或静态托管服务即可。

### 方式二：GitHub Pages
1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为发布源

### 方式三：Vercel / Netlify
1. 连接 GitHub 仓库
2. 自动部署，无需额外配置
