# PRD - 官网东方禅意极简风格

## 项目概述
官网美化：东方禅意极简风格，响应式适配PC和移动端。

## 设计规范

### 配色方案
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 宣纸白 | #FAF8F5 | 主背景色 |
| 墨黑 | #1A1A1A | 主文字色、强调元素 |
| 朱砂 | #C45C48 | 点缀色、CTA按钮、悬停状态 |

### 字体
- **主字体**：思源宋体 (Source Han Serif CN)
- **备选字体**：Noto Serif SC, serif

### 响应式断点
- **移动端**：< 768px
- **PC端**：≥ 768px

## 技术要求

### 动效
- 纯 CSS 动效，不使用 JavaScript 动画库
- 优先考虑性能和流畅度

### 响应式适配
- 移动优先设计原则
- 断点：768px
- 确保在各类设备上均有良好体验
